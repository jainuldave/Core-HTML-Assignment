1. HTML
    Primary Use Cases Key Features/Benefits
    Creating the structure and content of web pages
    • Simple and easy to learn.
    • Standardized language for web content.
    • Compatible with various browsers and devices.

  2. CSS
    Primary Use Cases Key Features/Benefits
    Styling and designing the visual presentation of web pages.
    • Separation of content and presentation.
    • Extensive styling capabilities.
    • Support for responsive and mobile-friendly design.

  3. JS
    Primary Use Cases Key Features/Benefits
    Adding interactivity and behavior to web pages.
    • Supports both front-end and back-end development (Node.js).
    • Rich ecosystem of libraries and frameworks (e.g., React, Angular, Vue).
    • Asynchronous programming for improved performance.
