1. <html>:
    1. The root element of an HTML document.
    2. Contains all other HTML elements on the page.
  2. <head>:
    1. Contains meta-information about the HTML document.
    2. Includes elements like <title>, <meta>, and <link>.
  3. <title>:
    1. Sets the title of the HTML document displayed in the browser's title bar or tab.
    2. Important for search engine optimization (SEO).
